# Day 2 – Backend Basics

## Goal

Set up a simple Express server with a health check endpoint.

## Steps

1. Install dependencies (express, nodemon, dotenv)
2. Create a `/health` endpoint
3. Use nodemon for development

## Concepts Learned

- Basics of Express.js
- Environment variables with dotenv
- Health check endpoints (important for monitoring)

Response:
{
"status": "ok",
"uptime": 10.234
}
